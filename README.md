# 🔌 JSONPlaceholder API Testing with Rest Assured

## 📖 Description
This project is an API automation testing framework for the JSONPlaceholder demo API using Rest Assured in Java.  
It validates RESTful services by testing different endpoints and verifying responses, status codes, and data.

The framework is designed to be scalable and maintainable for real-world API testing.

## 🚀 Features
- ✅ Automated API testing using Rest Assured
- 🔁 CRUD operations testing (GET, POST, PUT, DELETE)
- 🔍 Response validation (status code, headers, body)
- 🧪 Test execution using TestNG / JUnit
- 📊 Detailed logging for debugging
- ⚡ Clean and organized test structure

## 🛠️ Technologies Used
- Java
- Rest Assured
- TestNG / JUnit
- Maven
- JSON
